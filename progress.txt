- Workspace typechecks must cover every discovered `src` file without inherited or implicit exclusions, verified against TypeScript's parsed effective file set, and must reject all parsed configuration diagnostics while retaining effective `noEmit` with `outDir`/`declarationDir` outside `src`; domain source also rejects JavaScript and `@ts-nocheck`. Shared/domain TypeScript configuration cannot inherit or synthesize implicit module imports, and domain bare imports plus package-valued type/AMD directives must be Node built-ins, self-references, or declared dependencies without traversal segments and validated against forbidden runtime or type-package aliases.
